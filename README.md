# Proposal Roadmap

An interactive step-by-step guide for writing academic research proposals in Persian. This application assists graduate students through the complete process of thesis and dissertation proposal development.

## Features

- Nine comprehensive steps covering the entire proposal writing process
- Interactive progress tracking with visual indicators
- Full Persian language support with RTL layout
- Modern and accessible user interface
- Detailed educational content with practical examples

## Technology Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Lucide React
- Vazirmatn Font

## Installation
```bash
git clone https://github.com/naseridev/proposalroadmap.git
```
```bash
cd proposalroadmap
```
```bash
npm install
```
```bash
npm run dev
```

The application will be available at http://localhost:3000

## Deployment

This project is configured for GitHub Pages deployment:
```bash
npm run build
```

## Roadmap Steps

1. Smart Path Selection - Research topic selection strategies
2. Team Formation - Selecting advisors and supervisors
3. Scope Definition - Establishing research boundaries
4. Literature Review - Systematic review and gap identification
5. Innovation Architecture - Designing research methodology
6. Technical Writing - Formulating hypotheses and assumptions
7. Methodology - Defining tools and procedures
8. Ethics and References - Citation standards and plagiarism prevention
9. Defense Preparation - Final review and presentation guidelines
